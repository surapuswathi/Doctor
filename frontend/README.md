# Doctor Appointment Frontend

## Overview
This is the frontend application for the Doctor Appointment Booking System. It provides an intuitive user interface for patients, doctors, and administrators to manage appointments and healthcare services efficiently.

## Features
- User Registration & Login
- Doctor Registration & Login
- Doctor Listing
- Appointment Booking
- Appointment Management
- Patient Dashboard
- Doctor Dashboard
- Admin Dashboard
- Responsive User Interface

## Tech Stack
- React.js
- Vite
- React Router DOM
- Axios
- CSS

## Installation

bash npm install npm run dev

The frontend communicates with the backend REST APIs using Axios for:

- Authentication
- Doctor Management
- Patient Management
- Appointment Booking
- Dashboard Operations
